# Pomodoro-Timer-
## **使用工具:**

- **JQuery**
    - **localStorage**
    - **setInterval / clearInterval**
- **Pure CSS**
- **HTML**
## **功能:**

- **可以設定鬧鐘結束的語音提示**
- **工作/休息鬧鐘之間頁面顏色的轉換**
- **可以設定番茄鐘循環次數 [Hu](https://github.com/yuicheng1006/yui-portfolio)**
- **顯示累積工作時間**
- **顯示倒數時間**
